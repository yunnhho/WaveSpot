# WAVESPOT 진행 상황 요약 (2026-03-10 12:30 기준)

## ⚙️ 4. 백엔드 (NestJS) 코어 모듈 구축 진행 중
현재 **백엔드 기초 뼈대(Phase 1 앞부분)까지 구현**되어 있는 상태입니다.

- **공통 (Common):** 
  - JWT AuthGuard, ProGuard(구독 제어), AdminGuard 설정 완료
  - Public 데코레이터, API 응답 포맷 래퍼(ResponseInterceptor), 에러 포맷터(HttpExceptionFilter) 구현
- **설정 및 Prisma:**
  - `AppModule`에 설정 통합 및 `PrismaService`/`PrismaModule` 구성 및 글로벌 등록
- **인증 (AuthModule):**
  - Social 로그인 뼈대 구현 (카카오, 구글, 애플 분기 처리) - 현재는 키 미발급 상태로 Mock 로그인 동작
  - 빠른 테스트를 위한 개발용 로그인(`devLogin`) API 구현
  - JWT AccessToken 발급 및 Redis 기반 RefreshToken Rotation 로직 설계 적용 
- **유저 (UsersModule):**
  - `GET /me`, `PATCH /me`, `DELETE /me` (소프트 삭제) API 구현
