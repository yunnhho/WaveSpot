**2. 프로젝트 개요**

**2.1 서비스 소개**

WAVESPOT은 한국 해안 및 내수면의 수상 스포츠 스팟에 대한 기상·해양
데이터를 분석하여 종목별 활동 가능 지수를 즉시 제공하는 모바일 앱입니다.

입문자는 \'오늘 갈 수 있는가\'를 한눈에, 고급자는 기상 모델 수치와
관측소 실시간 데이터까지 깊이 있게 볼 수 있도록 같은 화면에서 Simple ↔
Expert 뷰를 전환할 수 있습니다.

**2.3 서비스 목표**

-   한국 유일의 수상 스포츠 특화 기상 앱

-   입문자: \'Simple View\'로 수치 없이 결과만

-   고급자: \'Expert View\'로 Windy급 데이터 심층 분석

-   공공 허가 데이터 기반 신뢰 스팟 DB + 유저 요청 시스템으로 지속 확장

-   Free / Pro 2단계 구독으로 전환 마찰 최소화

**3. UX 설계 원칙 (v1.2 갱신)**

**3.1 Simple ↔ Expert 뷰 토글 (v1.2 핵심)**

같은 스팟 상세 화면에서 우측 상단 토글 버튼 하나로 두 가지 뷰를
전환합니다. 온보딩에서 선택한 레벨이 기본값이 되며, 언제든지 설정에서
변경할 수 있습니다.

  ---------------------------------------------------------------------------
  **구분**      **Simple View (기본)**      **Expert View (Pro 전용)**
  ------------- --------------------------- ---------------------------------
  대상 사용자   입문자·일반인 (처음         중급\~고급자 (기상 지식 보유)
                시작하는 분)                

  활동 지수     등급 색상 + 자연어 문장     점수 숫자 + 세부 항목별 기여도 바
  표시          \'오늘 서핑하기 좋아요!     
                오후 2\~4시 최적\'          

  기상 수치     숨김 (결과만 표시)          풍속·파고·파주기·너울·수온·조류
                                            전체 수치

  주의사항      1줄 자연어 요약 \'바람이    항목별 경고 라벨 + 수치 범위 표시
                조금 강해요\'               

  예보 차트     일별 등급 색상 캘린더       시간별 통합 라인 그래프

  관측소 데이터 미표시                      예보값 vs 실측값 나란히 비교

  기상 모델     미표시                      GFS vs ECMWF 수치 비교
  비교                                      
  ---------------------------------------------------------------------------

⚠️ Expert View는 Pro 전용입니다. Free 사용자가 토글 시 구독 유도 팝업을
표시합니다.

**5.2 스팟 타입 정의 (v1.3 --- 마리나 통합)**

v1.2까지 별도 테이블로 분리되어 있던 마리나 데이터를 spots 테이블 내
spot_type 값으로 통합합니다. 마리나 고유 필드(계류 선석, 흘수, 요트 대여
등)는 spots.marina_info JSONB 컬럼에 저장합니다.

  -----------------------------------------------------------------------
  **spot_type 값** **설명**                     **marina_info 사용**
  ---------------- ---------------------------- -------------------------
  surfing          서핑 스팟 (해변·포인트)      ❌

  yacht            일반 요트·세일링 구역        ❌

  marina           마리나 (요트 계류·대여 시설  ✅ 마리나 전용 데이터
                   보유)                        

  waterski         수상스키·웨이크보드 구역     ❌
                   (내수면 포함)                

  kayak            카약·SUP 스팟                ❌

  diving           스쿠버다이빙·스노클링        ❌

  kitesurf         킷서핑·윈드서핑              ❌

  sup              스탠드업 패들보딩            ❌
  -----------------------------------------------------------------------

ℹ️ 하나의 스팟이 여러 타입을 가질 수 있습니다 (spot_types TEXT\[\]). 예:
양양 죽도 → \[\'surfing\', \'kayak\'\]. 마리나 스팟은 반드시
marina_info가 채워져야 합니다.

                       요트·다이빙 영향.

  AWS (방재기상관측망) 기상청 운영 전국 500여 개 자동기상관측소. 1분 단위
                       실측.

  해양관측부이         국립해양조사원 운영 해상 파고·수온·조류 실측 부표.

                       입문자용 뷰.

                       고급자용 Pro 뷰.

  Disclosure           접근하는 UX 원칙.

  Precompute           요청 전 서버에서 미리 계산해두는 방식. 응답속도
                       최적화.
